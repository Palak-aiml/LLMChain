# LLMChain Starter Project Templates

This directory contains ready-to-use templates and recipes for common LLMChain workflows:

- **qa_chain_template.py**: Retrieval-augmented question answering chain example.
- **summarization_chain_template.py**: Document summarization chain example.
- **conversational_chain_template.py**: Multi-turn conversational chain example.

## Usage

1. Copy the desired template to your project or run it directly.
2. Replace `YOUR_OPENAI_API_KEY` with your actual API key or configure your LLM provider.
3. Customize the chain, retriever, memory, or prompt manager as needed.

These templates are designed to help you get started quickly with LLMChain's advanced features.
